# Obsidian Homepage

启动 Obsidian 时自动展示自定义交互首页，集成日程管理、待办追踪、文件浏览与计时器。

## 主要功能

- 🏠 **自动首页** — 插件加载时自动打开首页，无需手动操作
- 👋 **智能问候** — 根据时段显示"早上好/上午好/中午好/下午好/晚上好"，名字可内联编辑，失焦自动保存
- 🕐 **实时时钟** — `HH:MM:SS` 格式，每秒刷新
- 📅 **日期与星期** — 右上角显示当前日期和星期
- 🧩 **组件化卡片** — 功能以卡片形式呈现，支持拖拽移动、右下角缩放，位置和尺寸持久化；卡片间自动碰撞避让；右侧侧边栏拖拽切换组件显隐

### 日程中心

- 📅 日历面板 — 月视图日历，点击日期切换选中，今日高亮
- ✅ 待办列表 — 添加/勾选/删除待办，四种颜色标记重要程度（高/中高/中/低），支持筛选和搜索
- 📊 统计柱状图 — 按颜色分组统计当天待办完成进度
- 🔄 昨日复盘 — 查看昨日未完成待办，支持逐项同步或一键全部同步到今天

### 待办列表

- 📋 多时间视图 — Tab 切换今天/本周/本月
- ⏱ 时间支持 — 添加待办时可设定开始时间和结束时间
- 📊 今日甘特图 — 纵向时间轴展示带时段的待办，彩色细条 + 事项文本与时长分行显示，已完成的保留在轴上（删除线标识）
- 📎 日程联动 — 与日程中心同时启用时，内容嵌入日程右侧面板替代原始待办区域，已完成事项移至统计栏下方

### 超级桌面

- 📁 多实例文件浏览器 — 可绑定 vault 下不同文件夹，网格展示内容
- 🏷 自定义标题 — 每个实例标题可内联编辑
- 📂 双击打开 — md 文件在 Obsidian 新标签页打开，其余类型用系统默认应用、图片使用系统默认应用打开、超链接使用默认浏览器打开

### 计时器

- ⏱ 双模式显示 — 表盘模式（圆形进度条）与数字模式（大字体倒计时）一键切换
- 🎛 滚动选择器 — 时/分/秒独立滚轮设定
- 🔔 到时提醒 — 倒计时归零弹窗通知

## 可用命令

| 命令 | 说明 |
|------|------|
| `打开首页` | 在当前标签页打开首页（若已打开则聚焦） |

## 安装

### 从源码构建

```bash
git clone <repo-url> obsidian-homepage
cd obsidian-homepage
npm install
npm run build
```

将以下三个文件复制到 vault 的 `.obsidian/plugins/homepage/` 目录：

- `main.js`
- `manifest.json`
- `styles.css`

在 Obsidian 设置 → 第三方插件中启用 **Homepage**。

### 开发

```bash
npm run dev     # watch 模式 + inline sourcemap，编译产物直接输出到 vault
npm run lint    # TypeScript 类型检查
```

配合 [hot-reload](https://github.com/pjeby/hot-reload) 插件可实现修改后自动重载。

## 使用方式

1. 启用插件后，Obsidian 启动时自动打开首页
2. 通过命令面板（`Cmd/Ctrl + P`）搜索"打开首页"可随时手动打开
3. 点击顶部问候语中的名字区域可编辑显示名称
4. 右侧侧边栏悬停展开，拖拽组件在"已添加"和"待添加"之间切换
5. 点击卡片任意非交互区域拖拽移动，拖拽右下角调整大小
